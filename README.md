# Retourne au Français

Retourne au Français modifie le texte des sites web pour récupérer un Français normal sans écriture dites 'inclusive'. 

## Installation de l'extension
Ouvrez Chrome et allez dans *chrome://extensions/*.<br>
Activez le "Mode développeur" en haut à droite.<br>
Cliquez sur "Charger l'extension non empaquetée" et sélectionnez le dossier contenant les fichiers de l'extension.

Une fois chargée, actualisez la page, le texte aura été modifié
